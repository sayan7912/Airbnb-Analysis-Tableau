# Airbnb Listing Analysis Dashboard

This Tableau project analyzes Airbnb listing data to uncover trends in pricing, occupancy, host performance, and customer review scores. The dashboard provides interactive visualizations and KPI metrics that help identify factors influencing listing success.

## Objectives
- Analyze Airbnb pricing patterns
- Compare Superhost and Non-Superhost performance
- Evaluate occupancy trends
- Identify key drivers of customer satisfaction
- Present actionable business insights through an interactive dashboard

## Tools Used
- Tableau
- SQL
- Excel/CSV

## Key Insights
- Superhosts consistently achieve higher review scores across all categories.
- Listing prices vary significantly across neighborhoods and property types.
- Occupancy rates differ based on listing characteristics.
- Host experience and activity influence overall performance.

## Dashboard Features
- KPI Cards
- Occupancy Analysis
- Pricing Analysis
- Host Performance Analysis
- Review Score Comparison
- Interactive Filters
